Ford GoBike Data Analysis¶

by Jozo Frankovic¶

Preliminary Wrangling¶

Ford GoBike is a regional public bicycle sharing system in the San Francisco Bay Area, California. Beginning operation in August 29th 2013 as Bay Area Bike Share, the Ford GoBike system currently has over 2,600 bicycles in 262 stations across San Francisco, East Bay and San Jose. This data set includes information about individual rides made in a bike-sharing system covering the San Francisco Bay area.

# Function for gathering data from URL and saving into the 'data' subdirectory directory

data = 'data' 

def gather_data(year, first_month, final_month, ind):

    for month in range(first_month, final_month + 1):
        month_str = str(month)
        year_str = str(year)
        month_dig = month_str.zfill(2)
        url = 'https://s3.amazonaws.com/baywheels-data/' + year_str + month_dig + ind + '-tripdata.csv.zip'
        response = get(url)  
        print(year, " - ", month)
        
        # Open the zip file, and save extracted data
    
        unzipped = ZipFile(BytesIO(response.content))
        unzipped.extractall(data)

Assess¶

Quality issues

  • start_time and end_time are objects rather than timestamps
  • bike_id, start_station_id, and end_station_id should be set to object
  • user_id should be set to category
  • bike_share_for_all_trip and rental_access_method can be dropped
  • Dataset can be enhanced by created columns for day, month, weekday, hour, year, etc., as well as duration in minutes
  • There are some nulls across the station information as aforementioned

# Function for ticks to be more easily read

def transform_axis_format(tick_val, pos):
    if tick_val >= 1000:
        val = int(tick_val/1000)
        return '{:d}K'.format(val)
    elif tick_val >= 1000000: 
        val = int(tick_val/1000000)
        return '{:d}M'.format(val)
    else:
        return int(tick_val)
